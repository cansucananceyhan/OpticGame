
In optics game the main purpose is to reach the light beam, that is put to a point by the user at the 
beginning of the game, to the target. In order to direct the light beam toward the target, straight mirrors are used as a
reflective material. Obstacles, which are placed on several places on the map, are provided for challenging the players.
Create a game option allows users to create their own game and unique levels. After players finish designing their game 
they can save it.
  
While developing this, JavaFX will be used in order to simplfy the design. JavaFX will also be used to easily create and
design user interfaces.
